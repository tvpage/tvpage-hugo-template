Inject Widget

You must manually place the widget in the three e-retailer PDPs

1. Install Custom JavaScript for websites (Google Chrome plugin )
Install from here

2. Visit one of the test pages or try with a new one
Walmart
QVC
HomeDepot

3. Click the CJS icon in your Google extensions

4. Copy and paste this code in the textbox (See Code to inject)

5. To toggle between widgets change the name of the widget in this variable: var widgetToTest = "carousel";. This example will only show carousel.
/*options: solo solo_cta carousel carousel_spotlight inline inline_spotlight sidebar*/

6. To choose the new version of the widgets make sure this variable is set to true: var testNew = true;. Setting it to false will summon the old ones

7. Finally, you must indicate in the code which page you are using. If you are pasting this code in the Walmart PDP, these variables will look like this: 

    	
    	//walmart.com 
    	var shortEl = "prod-Bot prod-PositionedRelative";
    	var largeEl = "topContent";
    	//qvc.com 
    	//var shortEl = "buyBoxPricing"; 
    	//var largeEl = "itemNo";

    	//homedepot.com 
    	//var shortEl = "buybox"; 
    	//var largeEl = "card__body scrollTouch";
Notice how the all the "var" in Walmart don't have "//". This means the widget will work on Walmart only. The other ones like WVC and Home Depot must have "//" in the beginning of the variable. Once you set up the code of the widget, reload the page and it should appear in the PDP
